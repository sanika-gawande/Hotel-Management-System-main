# Hotel-Management-System-main

Developed a web-based Hotel Management System using PHP and MySQL.

Designed to manage hotel operations such as room booking, customer registration, and payment records.

Implemented an admin panel to manage room details and booking information efficiently.

Built the frontend using HTML, CSS, and JavaScript for a simple and user-friendly interface.

Integrated MySQL database (bluebirdhotel) for storing and managing data.

Deployed and tested the project using XAMPP (Apache & MySQL) on localhost.

The system helps streamline hotel booking processes and improve overall management efficiency.
